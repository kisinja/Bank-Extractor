# Equity Bank Statement Customer Extractor (Hosted)

A simple web app that extracts customer details from Equity Bank statements:
- Phone numbers in **+254** format
- Customer names
- Total amount transacted (deduplicated)

## How to Use
1. Open the deployed link in any browser (phone or computer).
2. Upload statement images or PDFs.
3. Get a clean table + downloadable CSV.

Supports both text-based (digital) PDFs and scanned/image PDFs.

Deployed using Streamlit Community Cloud.
